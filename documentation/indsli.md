# 4. INDICES y SLICING

En la siguiente cadena de texto, podemos identificar cada caracter mediante un índice.
````python


palabra = "Python"
````

>"P" índice 0
>"y" índice 1
>"t" índice 2
>"h" índice 3
>"o" índice 4
>"n" índice 5

Si queremos referirnos a cualquier índice lo haremos de la siguiente manera entre corchetes []:
````python
print (palabra[3])
> 'h'
````
