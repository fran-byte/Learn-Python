# 8. SENTENCIAS ITERATIVAS (BUCLES)

## while

````python
c = 0
while c <= 5 : # Se ejecutará mientras c sea menor o igual a 5
    c+=1
    print("c vale: ",c)
print ("El bucle ha finalizado")
`````
